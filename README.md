## Project from classses
Stack: [Vue.js](https://vuejs.org/), [Express.js](https://expressjs.com/), [Node.js WebSocket library](https://github.com/websockets/ws).


## How to install
In 'server' folder run
```shell
    npm install
    npm start
```

In 'client' folder run
```shell
    npm install 
    npm run dev
```
